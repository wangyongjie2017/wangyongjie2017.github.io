变形和动画

  • 引言：目前来说，我们使用css3加一些js可以实现一些简单的动画，复杂的动画我们可以用html5中的”canvas”标签来实现。对于大多数非游戏类的站点来说，他们需要的就是简单动画。今天我们就来学习css3的变换和动画。今天主要内容：变换（transform）属性 + 渐变（transtion）效果 + 帧动画（@keyframes）效果。

  • (一) transform 变换

    • 作用：改变元素的尺寸 形状 位置等。
    • 兼容：Internet Explorer 10、Firefox 以及 【Opera 只支持2D】支持 transform 属性。
      Chrome 和 Safari 需要前缀 -webkit-。
Internet Explorer 9 需要前缀 -ms-。
    • 类型：
      • rotate 旋转 : rotate(360deg)
      • skew 扭曲 : skew(45deg,45deg) skewX()skewY() 没有Z
      • scale 缩放 : scale(.9,.9) scaleX()scaleY()
      • translate 位移 : translate(50px,50px) translateX()transformY() 取正值的时候：x向右y向下
    • transform-origin 允许你改变被转换元素的位置。
    • 3D：
      • z轴（ rotateZ() scaleZ() translateZ() translate3d(x,y,z) ）
      • perspective:100px;
      • transform-style:preserve-3d;

  • （二）transtion 过渡

    • 作用：顾名思义，渐进，某个属性样式改变 怎样改变，
      在CSS3引入过渡这个概念之前，CSS是没有时间轴的。也就是说所有的状态变化，都是即时完成
    • 兼容：ie9不支持 （Internet Explorer 10、Firefox、Opera 和 Chrome 支持 transition 属性。
      Safari 支持替代的 -webkit-transition 属性）
    • 4个参数：属性 过渡时间 渐进函数 延迟时间。
    • 渐进动画函数：ease（逐渐变慢 默认） linear（匀速） ease-in（加速） ease-out（减速）
      ease-in-out（加速然后减速）=贝塞尔曲线(0.42,0,0.58,1.0) 
cubic-bezier（自定义时间曲线） http://cubic-bezier.com/
steps(stepNumber,start/end);

  • （三）@keyframes 动画

    • 作用：是使元素从一种样式逐渐变化为另一种样式的效果。可以改变任意多的样式任意多的次数
    • 兼容：ie9不支持（Internet Explorer 10、Firefox 以及 Opera 支持。Chrome 和 Safari 需要前缀 -webkit-。）

    • 基本语法：

    • 可以用百分比来规定变化发生的时间，或用关键词from to = 0% 100%，为了浏览器兼容更好，我们尽量用百分比。
