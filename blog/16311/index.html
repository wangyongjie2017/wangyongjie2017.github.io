js中return的作用

前端 JavaScript 2017/01/18

本文于797天之前发表，文中内容可能已经过时。

1. 终止函数的继续运行

1. 无 return

2. 有 return

不加 return 的现象是先提示用户名没输入，然后提示密码没输入；加了return之后遇到一个没输入之后就不再继续检测

当返回的是 true 时，将继续操作。

当返回是 false 时，将中断操作。

当在 <a href="abc.html" onclick="return add_onclick()">Open</a> 中如果函数 add_onclick() 返回 true, 那么 页面就会打开 abc.html

2. 返回函数值

返回函数中的值，外部调用函数时，则会调用这个值。闭包中就是运用这个属性，调用函数内部的局部变量，以不影响全局变量。
